WooCommerce Multi Currency Report provide the sales report base on product sold currency.

== Description ==
<p>
Ni WooCommerce Multi Currency Report show the sales information base on different currency use in WooCommerce store. Report show the separate order total of each currency.
</p>
<h3>Dashboard Analysis </h3>
<p> Dashboard show the last order information, completed order count, registered and guest customer count.</p>

<h3>Order Report Information </h3>
<p>
Order report provide the order information like customer name, order details, currency details, payment method and customer billing information. 
</p>

<h3>Currency report Information </h3>
<p>
Currency report provide the summary report base on the product sold currency, report show the information order total, order tax shipping tax base on different currency.  
</p>
<p>
Group by the currency report base on currency, payment method and country. 
</p>

<h3>Filter and Search  </h3>
<p>
Report provide the option to filter the report base on currency and order period like today order, yesterday order and many more filter.
</p>

== Installation ==

1. Use the WordPress plugin installer or upload files to WordPress plugin directory.
1. Activate "Ni WooCommerce Multi Currency Report" plugin from admin panel.
1. Find "Currency Report" menu in admin.
